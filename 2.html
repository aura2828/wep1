기초 지식

노출(impression)

- 광고가 고객에게 보이는 것을 뜻함. 횟수가 기준.
- ex) A라는 사람에게 3회 보였을 때 -> 노출 3회

도달(Reach)

- 특정 광고 콘텐츠가 노출된 사람의 숫자. 보여진 사람이 기준
- ex) A라는 사람에게 3회 보였을 때 -> 도달 1회

CPM(Cost Per Mille)

- 1000회 노출당 비용
- CPM = (광고비용/노출수) x 1000
- oCPM(optimized CPM) : 페이스북의 기본 입찰방식(추천입찰방식+청구기준노출) 가장 행동가능성이 높은 유저에게 노출시키기 위해 최적화된 방식

CPC(Cost Per Click)

- 클릭당 비용. 디지털 광고에서 가장 흔하게 쓰이는 과금방식
- CPC = 광고비/클릭수

CTR(Click Through Rate)

- 노출대비 클릭율. 광고 콘텐츠의 반응률을 확인하기 위한 지표.
- CTR = (클릭수/노출수) x 100
- VTR(View Throught Rate) : 영상광고시청률 (페이스북에선 평균 10~15%가 일반적)

CPV(Cost Per View)

- 조회당 비용. 동영상광고에서 많이 사용되는 과금방식
- CPV = 광고비용/조회수

CPA(Cost Per Action)

- 사용자의 액션을 유도하는데 사용되는 비용
- CPA = 광고비용/액션수

CVR(Conversion Rate)

- 전환율. 광고를 통해 방문한 방문자가 발생시킨 전환(구매, 장바구니)활동의 비율
- CVR = 전환수/클릭수 x 100
- ex) 1000명이 게임 설치 광고를 봤고, 20명이 게임을 설치함. -> CVR=2% (20/1000*100)

CPI(Cost Per Install)

- 설치당 과금. App에서 주로 이용되는 과금방식

CPE(Cost Per Engagement)

- 참여당 과금.
- ex) 페이스북 좋아요, 공유 등

ROI(Return On investment)

- 광고 효율성

ROAS(return on ads spending)

- 광고비 대비 수익률
- 광고로 거둔 수익/광고비

LTV(LifeTime Value)

- 유저생애가치. 한 명의 유저가 서비스 진입부터 이탈(최종 활동)할 때까지 기간동안 생산해낸 가치(사용자당 기대 수익)
- 공식 : 유저당 평균 수입 x 1 / 이탈율

LAC(Lead Aquisition Cost)

- 리드 획득 비용.
- 공식 : 리드획득관련 전체 비용 / 신규 획득 리드 수

AARRR

- 해적지표. 데이브 맥클루어(Dave McClure)가 개발한 분석 프레임워크
- AARRR은 시장 진입 단계에 맞는 특정 지표를 기준으로 우리 서비스의 상태를 가늠할 수 있는 기준.
- Acquisition : 고객 확보 단계 (신규방문자, 가입자 확인)
Activation : 고객활동단계(평균체류시간, 페이지뷰 등 지표 확인)
Retention : 고객유지 / 재방문 단계(재방문율, 이탈경로 등 확인)
Revenue : 고객구매단계(매출 데이터 확인)
Referra : 고객추천단계(SNS공유수, 좋아요 수 등)
